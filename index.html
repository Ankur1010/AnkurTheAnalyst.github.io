Projects

Click Through Rate Prediction

Analysed and evaluated Click Through Rate (CTR) values to decide which unique member of the column has the highest influence. CTR is a method used to find out how frequently people clicked on the ad.

911 calls Data Analysis

I developed my interest in data analytics with this project and was surprised to see how data can be used to find valuable informations and insights.

Loan Repayment Prediction

Loan default prediction using deep learning considering features like current unpaid balance, loan tenure, unemployment rates, national interest rates and many other economical factors. The model performance was evaluated under various feature selection and architecture, where all comparisons were done using 5-fold cross validation.

Covid Data Exploration using SQL

Explored covid19 deaths and vaccination data using SQL. Answered various questions like likelihood of dying if you contract covid in your country.

Data Analysis for an Ecommerce company

Analyzed customer and user data to find out whether the company should focus on their mobile app or on their website. Moreover, we also built linear regression model with the accuracy of above 90% to predict yearly amount spent by the customers.
